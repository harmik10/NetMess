a lightweight LAN-based chat application for seamless communication in computer labs.

## Features
- Authentication
- Create Chat Room 
- Add New Users 
- View Currently Joined Users 
- Kick Users from Chatroom
- Update Chatroom Details
- Real-time Messaging
- Update Room and User Profile
